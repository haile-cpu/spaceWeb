# OrbitalX Landing Page

A **futuristic, responsive, startup-style landing page** for OrbitalX, an AI-powered orbital systems company. This project showcases modern web design techniques, including **grid-based layouts**, **glassmorphism**, **animated starfield backgrounds**, and **scroll reveal animations**.
